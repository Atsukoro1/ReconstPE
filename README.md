# ReconstPE

# Functionality
- Open any kind of x86_64 executable file and inspect it's contents
- GUI thanks to ImGui & Direct X11
- [WIP] Optional hex/ASCII view

# Why
I started this project to learn more about PE files and also write a little bit of C++

# Should I use this?
No, it's useless at it's current state - there are much better tools like PEBear
